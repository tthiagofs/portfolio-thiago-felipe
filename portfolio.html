Insightflow

Sistema web desenvolvido para automatização e análise de dados.


Criei esta plataforma para otimizar fluxos internos de trabalho. O Insightflow consome dados via API para gerar relatórios detalhados de performance do Meta Ads e Google Ads.

Além disso, o sistema compila painéis (dashboards) comparativos que facilitam a análise de resultados e a tomada de decisão estratégica pela equipe. É um projeto que materializa a união da minha visão estratégica de marketing com o desenvolvimento de software.
